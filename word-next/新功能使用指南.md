## 按音节搜索

### 💡 功能说明
通过输入音节（如 con, tion, ing），快速找到所有包含该音节的单词。

### 📝 使用步骤

1. **切换搜索模式**
   ```
   在搜索框上方，选择 "按音节搜索"
   ```

2. **输入音节**
   ```
   在输入框中输入音节，例如：con
   ```

3. **查看结果**
   ```
   系统会显示所有包含该音节的单词
   每页显示 50 个结果
   当前查询的音节会被高亮显示（紫色）
   ```

4. **翻页浏览**
   ```
   使用"上一页"和"下一页"按钮浏览更多结果
   ```

### 🎯 使用场景

✅ **学习音节规律**
```
搜索 "tion" - 查看所有 -tion 结尾的单词
了解这类单词的共同特征
```

✅ **词汇扩展**
```
搜索 "con" - 发现 conversation, consider, control 等
通过同音节单词扩展词汇量
```

✅ **发音练习**
```
搜索 "er" - 练习含有 er 音节的单词发音
集中练习特定音节
```

## 查看单词详情

### 💡 功能说明
点击单词可以查看完整的详细信息，包括自然拼读解析和词根词缀。

### 📝 使用步骤

1. **打开详情**
   ```
   点击任意单词卡片上的单词标题
   例如：点击 "conversation"
   ```

2. **查看完整信息**
   ```
   模态框会显示：
   - 📖 完整的单词信息
   - 📘 自然拼读解析（蓝色卡片）
   - 🟧 词根词缀（橙色卡片）
   - 📊 统计信息（查询次数、创建时间等）
   ```

3. **关闭详情**
   ```
   方法一：点击右上角的 ✕ 按钮
   方法二：点击模态框外的背景区域
   方法三：点击底部的"关闭"按钮
   ```

## 音节点击查询

### 💡 功能说明
点击任何单词卡片上的音节标签，立即查询包含该音节的所有单词。

### 📝 使用步骤

1. **找到音节标签**
   ```
   在任何单词卡片上，找到音节区域
   音节以蓝色标签形式显示
   ```

2. **点击音节**
   ```
   鼠标悬停在音节上会显示提示：
   "点击查询包含此音节的单词"
   
   点击感兴趣的音节
   ```

3. **查看结果**
   ```
   自动切换到"按音节搜索"模式
   搜索框自动填充该音节
   显示所有包含该音节的单词
   被点击的音节会高亮显示（紫色）
   ```

### 🎯 使用场景

✅ **快速探索**
```
查看 "conversation" 
→ 点击 "con" 音节
→ 发现 consider, control, continue...
```

✅ **关联学习**
```
学习 "action" 
→ 点击 "tion" 音节
→ 学习其他 -tion 结尾的单词
```

✅ **音节分析**
```
对比不同单词中的相同音节
理解音节在不同单词中的作用
```

## 使用技巧

### 💡 技巧 1: 组合使用功能

```
场景：深入学习包含 "con" 的单词

1. 搜索音节 "con"
   → 找到 20+ 包含 con 的单词

2. 点击 "conversation"
   → 查看详细的自然拼读和词根词缀

3. 在详情中点击 "tion" 音节
   → 发现更多 -tion 结尾的单词

4. 依次查看这些单词的详情
   → 系统化学习 -tion 词族
```

### 💡 技巧 2: 利用高亮功能

```
在音节搜索结果中：
- 当前查询的音节会用紫色高亮
- 其他音节保持蓝色
- 一目了然地看到目标音节的位置
```

### 💡 技巧 3: 高效浏览

```
单词列表 → 快速浏览
  ↓
发现感兴趣的音节 → 点击
  ↓
查看该音节的所有单词
  ↓
点击单词标题 → 查看详情
  ↓
在详情中点击其他音节 → 继续探索
```

### 💡 技巧 4: 学习路径

**路径 A：从音节开始**
```
1. 选择一个常见音节（如 ing, ed, tion）
2. 搜索该音节
3. 查看所有包含该音节的单词
4. 逐个点击查看详情，理解规律
```

**路径 B：从单词开始**
```
1. 浏览单词列表
2. 点击不熟悉的单词查看详情
3. 在详情中学习音节和词根
4. 点击感兴趣的音节继续探索
```

| 颜色 | 含义 | 使用场景 |

## 常见问题

### Q: 如何快速找到相关单词？
**A:** 使用音节搜索功能！输入音节比输入完整单词更快，而且能发现更多相关单词。

### Q: 详情模态框打不开？
**A:** 确保点击的是单词标题（蓝色粗体文字），不是整个卡片区域。

### Q: 音节高亮不明显？
**A:** 在音节搜索结果中，当前查询的音节会显示为紫色，与其他蓝色音节区分开来。

### Q: 搜索结果太多怎么办？
**A:** 使用分页功能，每页显示50个结果。也可以通过单词搜索缩小范围。

### Q: 如何清除搜索？
**A:** 点击搜索框旁边的"清除"按钮，或者直接修改搜索内容重新搜索。
