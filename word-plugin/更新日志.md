# 更新日志

## v1.0.0 (2025-11-04)

### 🎉 首次发布

#### ✨ 新功能

- 🖱️ **右键菜单添加单词**
  - 选中网页文本后右键快速添加
  - 自动提取英文单词
  - 支持全网页操作

- 🤖 **智能单词查询**
  - 集成 Deepseek AI 自动获取单词信息
  - 自动音节分词
  - 包含音标、翻译、音节

- 📱 **现代化弹窗界面**
  - 美观的渐变色设计
  - 清晰的信息展示
  - 响应式布局

- 🔐 **用户认证系统**
  - 支持用户注册和登录
  - JWT Token 安全认证
  - 本地存储登录状态

- 📊 **状态标记**
  - 新添加单词标记
  - 已存在单词显示查询次数
  - 直观的视觉反馈

- 🔍 **多种查询方式**
  - 右键菜单添加
  - 弹窗搜索框查询
  - 灵活便捷

- 💾 **数据同步**
  - 与后端服务实时同步
  - 跨设备数据共享
  - 安全可靠

#### 📦 包含文件

- `manifest.json` - 插件配置
- `popup.html` - 弹窗页面
- `popup.js` - 弹窗逻辑
- `styles.css` - 样式文件
- `api.js` - API 服务模块
- `background.js` - 后台脚本
- `content.js` - 内容脚本
- `generate_icons.py` - 图标生成脚本
- 完整的文档和使用指南

#### 📚 文档

- `README.md` - 完整功能说明
- `INSTALL.md` - 安装指南
- `快速开始.md` - 中文快速开始指南
- `test.html` - 测试页面
- `icons/ICON_GUIDE.md` - 图标制作指南

#### 🎨 设计特色

- 紫蓝渐变色主题
- 卡片式布局
- 彩色音节标签
- 流畅的动画效果

### 🔧 技术栈

- Manifest V3
- Vanilla JavaScript
- Chrome Extension APIs
- RESTful API 集成

### 🌐 支持的浏览器

- Chrome 88+
- Edge 88+
- 其他基于 Chromium 的浏览器
