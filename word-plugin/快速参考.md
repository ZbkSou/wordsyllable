# 🚀 快速参考卡片

## 📦 安装（3步）

```bash
# 1. 生成图标
cd word-plugin
python generate_icons.py

# 2. 加载插件
Chrome: chrome://extensions/
开启"开发者模式" → "加载已解压的扩展程序" → 选择 word-plugin

# 3. 启动后端
cd ..
双击 启动服务器.bat  # Windows
bash restart-backend.sh  # Linux/Mac
```

## 🎯 使用方式

| 方式 | 操作 |
|------|------|
| **右键添加** | 选中单词 → 右键 → "添加单词到记忆工具" |
| **直接查询** | 点击插件图标 → 输入单词 → 查询 |

## 🔑 API 地址

| 环境 | 地址 |
|------|------|
| 本地开发 | `http://localhost:5000` |
| 本地服务器 | `http://127.0.0.1:5000` |
| 远程服务器 | `http://your-server:5000` |

## 📊 显示内容

- ✅ 单词文本（大字显示）
- ✅ 音标（紫色斜体）

| 标记 | 含义 |
|------|------|
| 🟢 新添加 | 首次查询，已自动添加到词库 |

## 🐛 故障排查

| 问题 | 检查 |
|------|------|
| 插件加载失败 | icons 目录是否有三个图标文件 |
| 登录失败 | 后端服务是否运行（访问 /api/health） |
| 右键菜单无反应 | 刷新网页，重新加载插件 |
| Token 过期 | 退出重新登录 |
| 查询失败 | 检查 DEEPSEEK_API_KEY 配置 |

## 🔐 权限说明

| 权限 | 用途 |
|------|------|
| `contextMenus` | 创建右键菜单 |
| `storage` | 保存登录信息 |
| `activeTab` | 获取当前页面 |
| `host_permissions` | 访问后端 API |

## 📱 API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/auth/register` | POST | 用户注册 |
| `/api/auth/login` | POST | 用户登录 |
| `/api/words/lookup` | POST | 智能查询单词 |
| `/api/health` | GET | 健康检查 |
