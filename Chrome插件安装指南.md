# Chrome 插件安装指南

## 📦 插件位置

Chrome 插件位于：`word-plugin/` 目录

## 🚀 快速安装（3 步）

### 步骤 1：生成图标

进入 `word-plugin` 目录：

**Windows 用户：**
```bash
cd word-plugin
双击运行 "生成图标.bat"
```

**其他用户：**
```bash
cd word-plugin
python generate_icons.py
```

### 步骤 2：加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/` 并回车
3. 打开右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"**
5. 浏览并选择 `word-plugin` 文件夹
6. 点击"选择文件夹"

看到插件卡片出现即表示安装成功！✅

### 步骤 3：启动后端服务

回到项目根目录（`F:\Documents\11wordsyllable`）：

**Windows：**
```bash
双击 "启动服务器.bat"
```

**Linux/Mac：**
```bash
bash restart-backend.sh
```

验证服务：浏览器访问 http://localhost:5000/api/health

## ✨ 开始使用

### 首次登录

1. 点击 Chrome 右上角的插件图标（书本📚图标）
2. 在登录界面输入：
   - API 地址：`http://localhost:5000`
   - 用户名和密码
3. 如果没有账号，点击"注册账号"

### 添加单词

**方式 1：右键菜单**
1. 在任意网页选中英文单词
2. 右键 → "添加单词到记忆工具"
3. 查看弹窗中的单词详情

**方式 2：直接搜索**
1. 点击插件图标
2. 在搜索框输入单词
3. 点击"查询"

## 🐛 常见问题

### 1. 插件加载失败

**错误**：缺少图标文件

**解决**：
- 运行 `生成图标.bat` 或 `python generate_icons.py`
- 或使用任意图片临时替代（重命名为 icon16.png、icon48.png、icon128.png）

### 2. 无法连接服务器

**错误**：登录失败或网络错误

**解决**：
- 确认后端服务正在运行
- 访问 http://localhost:5000/api/health 测试
- 检查 API 地址配置

### 3. 右键菜单没反应

**解决**：
- 刷新网页
- 在 chrome://extensions/ 重新加载插件
- 确认已登录

### 4. Token 过期

**解决**：
- 点击退出登录
- 重新登录即可
